# Nodos y Arcos - Proyecto Final POO (2023)

Proyecto final desarrollado para la asignatura **Programación Orientada a Objetos (POO)** durante el año **2023**.

## Descripción

Esta aplicación web permite crear, editar y administrar grafos dirigidos de forma interactiva mediante un lienzo (Canvas de HTML5). El usuario puede agregar nodos, conectar nodos mediante arcos, asignar pesos y modificar la estructura del grafo en tiempo real.

El objetivo principal del proyecto fue aplicar los conceptos fundamentales de Programación Orientada a Objetos, diseño de estructuras de datos y manipulación de gráficos utilizando tecnologías web.

---

## Características

* Crear nodos.
* Crear nodos fuente.
* Crear nodos sumidero.
* Crear arcos dirigidos entre nodos.
* Asignar peso a cada arco.
* Editar el ID y el tamaño de los nodos.
* Modificar el peso de los arcos.
* Mover nodos libremente.
* Eliminar nodos y arcos.
* Limpiar completamente el canvas.
* Guardar el grafo en un archivo.
* Cargar grafos previamente guardados.

## Instalación

1. Clonar el repositorio:

```bash
git clone https://github.com/Kenner07/Nodos-y-Arcos-Proyecto-POO.git
```

2. Acceder al directorio del proyecto:

```bash
cd Nodos-y-Arcos-Proyecto-POO
```

3. Abrir el archivo `index.html` en un navegador o ejecutar el proyecto utilizando una extensión como **Live Server** en Visual Studio Code.

---

## Uso

Desde el panel lateral de la aplicación es posible:

* Agregar nodos.
* Agregar nodos fuente y sumidero.
* Crear arcos entre nodos.
* Actualizar las propiedades de nodos y arcos.
* Mover nodos dentro del lienzo.
* Eliminar nodos y arcos.
* Guardar el grafo en un archivo.
* Cargar un grafo previamente almacenado.
* Limpiar completamente el área de trabajo.
